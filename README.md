SignalDesk Africa is a premium media intelligence and creator monetisation platform built specifically for the South African market. It solves the "global tool gap" by providing deep local context, ZAR pricing, and POPIA compliance.

## 🚀 The Vision

Most media monitoring tools are built for US/EU markets. SignalDesk is built for **Cape Town, Johannesburg, and Durban**. It provides a shared backend intelligence layer that powers two distinct products:

1.  **SignalDesk Pro**: A professional dashboard for PR agencies and corporate communication teams.
2.  **Creator Platform**: An AI-powered assistant helping SA content creators maximise their Facebook Bonus earnings.

## ✨ Key Features

- **Real-time Data Ingestion**: Monitoring X, TikTok, Instagram, Facebook, and SA news cycles every 15 minutes.
- **Local Sentiment Engine**: Custom NLP models trained on SA dialects and entities (brands, politicians, public figures).
- **AI Analyst Reports**: Automated generation of brand health reports and creator "attack plans".
- **WhatsApp Alerts**: Real-time crisis detection and deadline reminders delivered via WhatsApp Business API.
- **ZAR Native**: Predictable local pricing with no USD/EUR currency fluctuation risk.

## 🛠 Tech Stack

- **Frontend**: Semantic HTML5, Vanilla CSS3 (Custom Design System), and Modern JavaScript (ES6+).
- **Animations**: Intersection Observer for scroll reveals and HTML5 Canvas for the animated "Signal" background.
- **Typography**: Syne (Heads) & DM Sans (Body) via Google Fonts.
- **Lead Capture**: FormSubmit integration for the launch waitlist.

## 🏃 Getting Started

To run the landing page locally:

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/ZolileN/signaldesk-page.git
    ```

2.  **Serve the files**:
    You can use any local server. For example, using `npx`:
    ```bash
    npx serve
    ```

3.  **Open the browser**:
    Navigate to `http://localhost:3000` (or the port specified by your server).
